# 🔋 AIco Buddy (AI + Eco Buddy)

> **"생성형 AI, 이제 더 똑똑하고 친환경적으로 사용하세요."**  
> **AIco Buddy**는 ChatGPT, Gemini, Claude Web 환경에서 사용자가 입력하는 프롬프트와 생성되는 응답의 토큰 수를 실시간으로 계산하고, 그에 따른 전력 소모량, 탄소 배출량, 냉각수 소비량 등의 디지털 환경 발자국을 온디바이스(On-device)로 정밀 측정하여 시각화해 주는 Chrome 확장 프로그램입니다.

---

## 💡 주요 기능

1. **실시간 온디바이스(On-device) 토큰 측정**
   * 외부 서버 통신 없이 웹 브라우저 내에서 직접 한글과 영어의 언어적 특성을 고려한 고성능 토큰 추정 알고리즘 탑재.
2. **다차원 디지털 탄소 발자국 분석**
   * 입력 필드 작성 시 실시간으로 예상 전력(Wh), 냉각수(mL), 탄소 배출량(g CO₂)을 계산하여 시각적 경고 및 요약 제공.
3. **환경 영향 직관적 환산 (Equivalents)**
   * 복잡한 수치를 "스마트폰 완충 횟수", "LED 전구(9W) 점등 시간", "소나무 한 그루의 정화 소요 일수"로 변환하여 사용자가 낭비 수준을 체감하도록 유도.
4. **프롬프트 압축 및 최적화 제안**
   * 한국어/영어의 불필요한 미사여구(인사말, 중복 요청 단어 등)를 제거하는 프롬프트 압축 제안을 통해 불필요한 토큰 낭비 방지.
5. **난이도 기반 모델 라우팅 조언 (SLM vs LLM)**
   * 질문의 복잡도를 분석하여 가벼운 질문은 소형 모델(SLM, 예: Gemini Flash, GPT-4o-mini), 고난도 질문은 대형 모델(LLM, 예: Gemini Pro, GPT-4o)로 사용하도록 행동 양식 가이드.
6. **중복 질문 및 에너지 낭비 방지**
   * 최근 질문 내역의 해시(Hash) 캐싱을 통해 동일한 질문의 중복 유입 및 중복 추론으로 발생하는 전력 낭비 차단.

## 🔬 환경 자원 소모량 산출 근거 및 공식

현재 소스 코드([tokenizer.js](file:///Users/ckyac/workspaces/aicobuddy/lib/tokenizer.js) 및 [analyzer.js](file:///Users/ckyac/workspaces/aicobuddy/lib/analyzer.js))에 구현된 환경 자원 소모량 산출 기준은 다음과 같이 설정되어 있습니다.

### 1. 토큰 추정 알고리즘
* **한국어 (한글)**: 1글자당 **1.8 토큰**
* **영어**: 단어당 **1.25 토큰** (8자 초과 단어는 4글자당 1토큰씩 추가)
* **공백**: 개당 **0.6 토큰**
* **특수문자 및 이모지**: 개당 **0.8 토큰**

### 2. 에너지 소모량 (전력) 산출 공식
* **대형 모델 (LLM)**: 기본 **0.5 Wh** + 토큰당 **0.0002 Wh**
* **소형 모델 (SLM)**: 기본 **0.05 Wh** + 토큰당 **0.00002 Wh**

### 3. 탄소 배출량 (Carbon Footprint) 산출 공식
* **탄소 배출량(g CO₂)** = 전력 사용량(Wh) $\times$ **0.475**

### 4. 냉각수 소비량 (Water Footprint) 산출 공식
* **대형 모델 (LLM)**: 기본 **2 mL** + 토큰당 **0.01 mL**
* **소형 모델 (SLM)**: 기본 **0.2 mL** + 토큰당 **0.001 mL**

### 5. 환경 파괴 지표 환산 기준 (Equivalents)
* **스마트폰 완충 횟수**: 탄소 배출량(g CO₂) / **5.0**
* **LED 전구 작동 시간**: 탄소 배출량(g CO₂) / **4.3**
* **소나무 정화 일수**: 탄소 배출량(g CO₂) / **60.0**
* **종이컵 환산**: 냉각수 소비량(mL) / **250.0**

> [!WARNING]  
> 현재 설정되어 있는 위 산출 공식과 수치들은 일부 학술 자료 및 임의 기준을 기반으로 작성되었습니다. 따라서 구체적인 실증 근거가 준비되지 않아 향후 보완이 필요합니다.

## 🛠 설치 방법

1. 이 저장소를 로컬 컴퓨터로 복제(Clone)하거나 Zip 파일로 다운로드합니다.
2. Google Chrome 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 이동합니다.
3. 우측 상단의 **개발자 모드 (Developer mode)** 토글 스위치를 켭니다.
4. 좌측 상단의 **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 버튼을 클릭합니다.
5. 다운로드한 프로젝트 폴더(내부에 `manifest.json`이 있는 디렉터리)를 선택하여 등록합니다.
6. Chrome 브라우저 툴바에서 확장 프로그램 아이콘을 고정(Pin)하여 실시간 모니터링 대시보드를 사용할 수 있습니다.
